# StudyBuddy

An AI-powered learning platform that combines intelligent tutoring, quiz generation, adaptive flashcards, and document summarization to help students study effectively in one unified interface.

## The Problem

Students struggle with fragmented study tools, ineffective learning methods, and lack of adaptive feedback. Time spent organizing materials often exceeds actual study time, while passive learning approaches lead to poor retention.

## The Solution

StudyBuddy consolidates essential study tools powered by AI into one intuitive platform, enabling students to:

- **Get Instant Answers** - Interactive AI tutoring with contextual understanding and multi-turn conversations
- **Generate Practice Materials** - AI-powered quizzes with customizable difficulty and explanations
- **Study Smarter** - Intelligent flashcards using spaced repetition algorithms for optimal retention
- **Extract Insights** - Upload PDFs and get AI-generated summaries with key points
- **Track Progress** - Monitor learning with streaks, statistics, and weekly activity visualization

---

## Key Features

**AI Chat Tutoring** - Real-time conversational learning with session history and contextual responses

**Intelligent Quiz Generator** - Multiple-choice quizzes on any topic with instant scoring and detailed explanations

**Spaced Repetition Flashcards** - SM-2 algorithm automatically schedules reviews based on performance for optimal memory retention

**Document Summarization** - Extract key insights from PDFs instantly with AI-powered analysis

**Learning Dashboard** - Track study streaks, quiz scores, and weekly activity to stay motivated

**Global Search** - Find previous learnings across all study materials instantly

**Secure Authentication** - User accounts with JWT-based security and per-user data isolation

---

## Tech Stack

**Frontend:** React 19, Tailwind CSS, Shadcn/UI Components, Recharts for Analytics

**Backend:** FastAPI, MongoDB, JWT Authentication, PDF Processing

**AI Integration:** Large Language Model APIs for intelligent content generation

---

## Architecture Overview

The application follows a modern full-stack architecture with a React-based frontend communicating with a FastAPI backend. All user data is isolated and secured through JWT authentication, while AI capabilities enable dynamic content generation for quizzes, flashcards, and document summaries.
